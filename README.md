# SQL Analysis of Digital Payment Transactions

This project demonstrates the use of SQL to analyze
digital payment transaction data.

## Tools
- SQL
- MySQL Workbench

## Key Analysis
- Total number of transactions
- Total amount transferred per user
- Average transaction value
- Currency-based filtering
- High-value user identification

## Dataset
Synthetic data created for learning and portfolio purposes.
